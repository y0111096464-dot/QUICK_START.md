## 🔧 التعديلات المتقدمة

### إضافة كورس جديد برمجياً:
```javascript
const newCourse = {
    id: 16,
    name: 'اسم الدورة',
    instructor: 'اسم المدرب',
    description: 'وصف الدورة',
    level: 'beginner', // or intermediate, advanced
    lessons: 20,
    hours: 40,
    students: 0,
    category: 'web', // web, games, backend, ai, databases
    content: [
        { title: 'الدرس 1', description: 'الوصف' }
    ],
    requirements: ['متطلب 1', 'متطلب 2'],
    outcomes: ['النتيجة 1', 'النتيجة 2'],
    skills: ['مهارة 1', 'مهارة 2']
};
app.courses.push(newCourse);
app.saveCourses(app.courses);
```

### تغيير الألوان:
عدّل في `css/academy.css`:
```css
:root {
    --primary: #6366f1;
    --secondary: #10b981;
    --accent: #f59e0b;
}
```
