# منصة تعليمية متكاملة - اكاديمية البرمجة

## 📚 وصف المشروع

منصة تعليمية حديثة وشاملة لتعليم البرمجة والأكواد. توفر المنصة:
- **كورسات مجانية ومدفوعة** في مختلف لغات البرمجة
- **نظام تسجيل دخول وإدارة حسابات** آمن
- **لوحة تحكم إدارية** متقدمة للإداريين
- **تجربة تعليمية تفاعلية** وحديثة
- **واجهة مستخدم جميلة وسهلة الاستخدام**

## 🚀 المميزات الرئيسية

### 1. **المستخدمون العاديون**
- ✅ التسجيل والدخول الآمن

- ✅ شراء الكورسات المدفوعة
- ✅ عرض ملف شخصي مع الكورسات المسجلة
- ✅ البحث والتصفية عن الكورسات

### 2. **الإدارة (Admin)**
- 🔐 دخول آمن باستخدام كلمة مرور (2006)
- 📊 لوحة تحكم شاملة مع إحصائيات
- ➕ إضافة كورسات جديدة
- ✏️ تعديل وحذف الكورسات
- 👥 عرض وإدارة المستخدمين
- 📋 عرض الكورسات المسجلة لكل مستخدم

## 🎯 كيفية الاستخدام

### للمستخدم العادي

1. **الدخول للموقع**
   - اضغط على "تسجيل الدخول" في القائمة العلوية

2. **إنشاء حساب جديد**
   - اختر "إنشاء حساب"
   - أدخل بيانات: الاسم، البريد الإلكتروني، كلمة المرور

3. **الاشتراك في الكورسات**
   - اذهب إلى قسم "الكورسات"
   - اختر كورس من الكورسات المجانية واشترك
   - أو اشترِ كورس مدفوع

4. **عرض ملفك الشخصي**
   - انقر على اسمك في الأعلى
   - شاهد الكورسات المسجلة

### للإدارة (Admin)

1. **الدخول إلى لوحة التحكم**
   - اضغط على "تسجيل الدخول"
   - أدخل: admin@academy.com و كلمة المرور: 2006

2. **إدارة الكورسات**
   - اذهب إلى "إدارة الكورسات"
   - أضف كورس جديد
   - عدّل أو احذف الكورسات

3. **مراقبة المستخدمين**
   - اذهب إلى "المستخدمون"
   - عرض جميع المستخدمين المسجلين
   - عرض تفاصيل كل مستخدم

4. **عرض الإحصائيات**
   - في لوحة التحكم الرئيسية
   - شاهد عدد المستخدمين والكورسات

## 🔐 الأمان

- البيانات محفوظة في localStorage
- كلمة مرور قوية للإدارة (2006)
- لا يمكن الوصول لصفحات الإدارة بدون دخول

**ملاحظة:** في بيئة الإنتاج، يجب استخدام:
- قاعدة بيانات حقيقية (MongoDB، MySQL، إلخ)
- ترميز البيانات (Encryption)
- تحقق من الجلسات (Sessions)
- استخدام HTTPS

## 📝 ملاحظات مهمة

1. **البيانات محلية**: جميع البيانات تُحفظ في localStorage الخاص بالمتصفح
2. **لا توجد قاعدة بيانات**: هذا مشروع تعليمي أولي
3. **التطوير المستقبلي**: يمكن ربط قاعدة بيانات حقيقية والدفع الإلكتروني

## 🎓 كيفية التطوير والإضافة

### إضافة كورس جديد برمجياً

```javascript
const newCourse = {
    id: 7,
    name: 'اسم الكورس',
    instructor: 'اسم المدرب',
    description: 'وصف الكورس',
    type: 'free', // أو 'paid'
    price: 0, // السعر للكورسات المدفوعة
    level: 'beginner', // beginner, intermediate, advanced
    lessons: 10,
    students: 0,
    rating: 4.5,
    content: [
        { title: 'الدرس 1', description: 'الوصف' },
        // ... المزيد من الدروس
    ]
};

app.courses.push(newCourse);
app.saveCourses(app.courses);
```

### تعديل أنماط التصميم

```css
    --danger-color: #ef4444;       /* الأحمر للحذف */
    /* ... */
}
```
