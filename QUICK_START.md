## 💾 البيانات والتخزين

جميع البيانات يتم تخزينها في **localStorage** (الذاكرة المحلية)

**المميزات:**
- البيانات محفوظة حتى بعد إغلاق المتصفح
- لا حاجة لخادم أو قاعدة بيانات
- سريع جداً وآمن نسبياً

**الملاحظات:**
- كل متصفح له بيانات منفصلة
- مسح السجل قد يحذف البيانات
- استخدم F12 لرؤية البيانات المخزنة

## 🔧 التطوير والتخصيص

### إضافة كورس جديد برمجياً:

```javascript
const newCourse = {
    id: 7,
    name: 'اسم الكورس',
    instructor: 'اسم المدرب',
    description: 'وصف الكورس',
    type: 'free', // أو 'paid'
    price: 0,
    level: 'beginner', // beginner, intermediate, advanced
    lessons: 10,
    students: 0,
    rating: 4.5,
    content: [
        { title: 'الدرس 1', description: 'الوصف' }
    ]
};
app.courses.push(newCourse);
app.saveCourses(app.courses);
```

### تغيير الألوان:

عدّل في `css/styles.css`:
```css
:root {
    --primary-color: #2563eb;    /* الأزرق */
    --secondary-color: #10b981;  /* الأخضر */
    --danger-color: #ef4444;     /* الأحمر */
}
```
