# منصة التعليم المتكاملة - إرشادات المشروع

## نظرة عامة
منصة تعليمية حديثة لتعليم البرمجة مع نظام كامل للكورسات والمستخدمين والإدارة.

## الميزات الرئيسية
- ✅ نظام تسجيل دخول آمن
- ✅ كورسات مجانية ومدفوعة
- ✅ لوحة تحكم إدارية متقدمة
- ✅ إدارة المستخدمين
- ✅ واجهة مستخدم حديثة وسهلة الاستخدام

## المجلدات والملفات
```
├── index.html           - الصفحة الرئيسية

├── js/
│   ├── app.js          - البرنامج الرئيسي

└── README.md           - التوثيق الكامل
```
